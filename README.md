# ClubCrest Showcase

A full-stack web application for college club discovery and management, built with React, TypeScript, and Supabase.

## Features

- Browse and explore college clubs with a dynamic showcase interface
- Role-based access with a hidden admin panel for club management
- Real-time data powered by Supabase (PostgreSQL backend)
- Responsive UI with a custom crimson red design theme
- Deployed live on Vercel

## Tech Stack

**Frontend:** React, TypeScript, Tailwind CSS, Vite  
**Backend:** Supabase (PostgreSQL + Auth)  
**Deployment:** Vercel
